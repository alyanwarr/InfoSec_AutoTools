# On Unix:  

Enter command `nc -l ip port`

# On Windows: 

Enter command `telnet ip port`

Where `ip` is the unix machine private `ip` and `port` is any port number.

# Example: 

`nc -l 192.168.1.100 6666`

`telnet 192.168.1.100 6666`
